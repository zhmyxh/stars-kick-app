# StarsKick App Project

Git project of Telegram Mini-App "StarsKick"
// Only Frontend

## Technologies

-   React
-   JavaScript
-   CSS
-   Zustand, ReactQuery

## Install & Launch

### 1. Clone repository

``` bash
git clone https://github.com/username/project-name.git
cd project-name
```

### 2. Instal dependencies

``` bash
npm install
```

or

``` bash
yarn
```

### 3. Launch dev mode

Vite:

``` bash
npm run dev
```

App will be availible for adrress:

-   http://localhost:5173

## Project build

``` bash
npm run build
```
